#### 0.1.0

> 2 April 2026

- Initial release
- DNS Lookup node — raw DNS queries over the wire protocol (RFC 1035)
- DNS Watch trigger node — poll for DNS record changes
- 13 record types: A, AAAA, MX, TXT, SRV, PTR, CAA, SOA, NS, CNAME, NAPTR, DNSKEY, TLSA
- Resolver modes: well-known, custom, authoritative (auto-discover), system
- TXT record parsing: SPF, DMARC, DKIM, verification tokens
- Output consistency check for multi-resolver propagation monitoring
- Zero external runtime dependencies — pure DNS wire protocol implementation
